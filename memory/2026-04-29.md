## Usage Examples

### Build GSMUI Component Library
```bash
python3 scripts/ui_library_builder.py \
  --frameworks gsmui \
  --components card button input modal comment \
  --theme gsmui \
  --output ./gsmui-library
```

### Monitor Token Issues
```bash
python3 scripts/token_burn_monitor.py \
  --project /path/to/laravel \
  --scan  # or --fix
```

### Install Laravel MCP Package
```bash
python3 scripts/install_laravel_mcp.py \
  --create \
  --name my-laravel-server \
  --output ./my-package
```
