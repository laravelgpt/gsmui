

### Composition Pattern
```php
// Build complex UIs from simple components
<Card>
    <CardHeader>
        <Heading>Title</Heading>
    </CardHeader>
    <CardBody>
        <DataTable :data="items" />
    </CardBody>
    <CardFooter>
        <Pagination :items="items" />
    </CardFooter>

```

### Slot System
```php
// Dynamic content injection
<Alert>
    <template #icon>
        <Icon name="warning" />
    </template>
    <template #default>
        Warning message
    </template>
    <template #action>
        <Button>Dismiss</Button>
    </template>
</Alert>
```
